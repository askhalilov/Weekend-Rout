# Маршрут выходного дня — Vue/Vite

Версия review5: многостраничный travel-сервис по Москве.

## Что добавлено

- Отдельная страница «Мой маршрут».
- Автоподбор маршрута по времени и настроению.
- Drag-and-drop порядок точек в маршруте.
- Расчёт общего времени: посещение, переезды, паузы.
- Рекомендации кафе, ресторанов, отдыха, метро и фото-точек.
- Режимы карты: маршрут / еда / отдых / метро / фото-точки.
- Расширенный журнал с фактами, статьями, идеями и новостями проекта.
- Пустой маршрут по умолчанию: пользователь сам собирает сценарий.

## Запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```

## Backend для сдачи

В проект добавлен простой Node.js backend. Подробности — в файле `README_BACKEND.md`.

Быстрый запуск frontend и backend вместе:

```bash
npm install
npm run dev:full
```

Проверка API:

```text
http://localhost:3001/api/health
```
